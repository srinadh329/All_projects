Language translation

Multilinguality
Our agent is built on a multilingual support, meaning that you can speak several languages with the same bot. Each time a new user tries the chatbot, the developer can detect what language preference they have and invoke the correct language in the chatbot.
Multilingual synonym package
Our platform supports synonyms of multiple languages without having to rebuild the bot definitions. You can select the bot in your preferred language and then enable one or more supported languages. As part of enabling the language, the platform automatically translates all standard responses and you can then translate various other bot elements in the new language.
