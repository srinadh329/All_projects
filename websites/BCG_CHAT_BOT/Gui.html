Dashboard with customizable overview
Dashboard visualizes the performance of the bot and the behaviour of the customers interacting with bot. It also facilitates the user interaction for the developers to train the bot.
Agent management
GUI enables the admin to add the handover agents and can view the performance of the agents from the feedback given to them from the customers.
Audit log
In GUI we can visualize the sequence of actions performed by the bot and the agents with the customers in a customizable overview for a better understanding of the tasks performed in the platform.
